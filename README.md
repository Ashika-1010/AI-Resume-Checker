# 🚀 Sharply – AI Resume Checker

An AI-powered resume analysis platform built with the **MERN Stack** and **Google Gemini AI**. Sharply helps users improve their resumes through ATS analysis, AI-powered rewrite suggestions, resume versioning, and interactive analytics.

---

## ✨ Features

* 📄 Upload PDF resumes
* 🤖 AI-powered resume parsing
* 📊 ATS score analysis
* 🎯 Keyword matching & missing keyword detection
* ✍️ AI-generated resume rewrite suggestions
* 📈 Resume version history & comparison
* 📊 Analytics dashboard
* 📚 Resume insights
* 📤 Resume export

---

## 🛠 Tech Stack

**Frontend**

* React.js
* Vite
* Tailwind CSS
* Axios

**Backend**

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

* Multer

**AI**

* Google Gemini AI

## 🚀 Getting Started

```bash
git clone https://github.com/Ashika-1010/AI-Resume-Checker.git

cd AI-Resume-Checker

# Backend
cd backend
npm install

# Frontend
cd ../frontend/ui
npm install
```

Create a `.env` file in the backend using the provided `.env.example`, then run:

```bash
# Backend
npm run dev

# Frontend
npm run dev
```
